## About the project:

**Color Mix**  

This web app offers a hands-on way to explore HTML canvas by mixing colors in an engaging, magical setting.  

- **Color Creation**: Combine primary colors (red, blue, yellow) to form secondary colors in a dynamic, interactive cauldron.  
- **Immersive Experience**: A visually captivating interface enhances the learning process.  

Dive into the canvas and unlock the art of color mixing!
